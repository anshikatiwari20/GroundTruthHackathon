## Setup Instructions

1. **Clone the Repository-**
First, clone the repository to your local machine by running:
   ```bash
    git clone https://github.com/anshikatiwari20/GroundTruthHackathon.git

    ```

2. **Navigate to the Project Folder-**
After cloning the repository, navigate to the project directory by running:
    ```bash
    cd GroundTruthHackathon

    ```

3. **Install Dependencies-**
If you haven't already installed the required dependencies, run the following command:
    ```bash
    pip install flask pandas matplotlib openpyxl sqlalchemy html5lib lxml
    pip freeze > requirements.txt
    ```
This will install all the necessary packages listed in the package.json file.

4. **Start the Server-**
To start the server, run:
    ```bash
    python app.py

    ```
The server will be running locally on http://127.0.0.1:5000.
